## 5. 将两数的**相乘操作**转化为**位运算和加法**
简单来说就是将一个数转为二进制形式，用2的幂的和来替代这个数，然后另一个数分别乘以2的幂。

比如，`12 * 13`：
```
13(1101) = 2^3 + 2^2 + 2^0

12 * 13 = 12 * ( 2^3 + 2^2 + 2^0 )
        = (12 << 3) + (12 << 2) + (12 << 0)
```

这也被叫做俄罗斯民间乘法，具体可以看这篇文章：[俄罗斯农民乘法](https://blog.csdn.net/iteye_4501/article/details/81682160)。
